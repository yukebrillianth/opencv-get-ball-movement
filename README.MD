## How to Use
### 1. Installation
Clone this repo
```bash
git clone https://github.com/yukebrillianth/opencv-get-ball-movement.git ball-position

cd ball-position
```

Make a build directory
```bash
mkdir build
cd build
```

run cmake
```bash
cmake ..
```

### 2. Usage
compile before run the program
*(run in build directory)*
```bash
make
```
run the program
```bash
./BallMovement
```
